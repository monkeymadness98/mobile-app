A simple Expo app for daily mood, energy, and stress tracking.


## How to Run
1. Clone this repository
2. Install dependencies: `npm install` or `yarn install`
3. Start Expo: `expo start`
4. Open in **Expo Go** on iOS or Android


## Dependencies
- react-native
- @react-navigation/native
- @react-navigation/stack
- @react-native-community/slider


## Features
- Daily mood, energy, stress input
- Weekly averages
- Simple, clean UI
